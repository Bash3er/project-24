# Course Management System

An Angular application for managing courses and student enrollments.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the mock backend (in one terminal):
   ```bash
   npm run json-server
   ```

3. Start the app (in another terminal):
   ```bash
   ng serve -o
   ```

The app will open automatically at `http://localhost:4200`

## Features

- Course listing and details
- Student management
- Enrollment tracking
- Dashboard overview

## Tech Stack

- Angular 17
- Angular Material
- JSON Server (mock backend)
