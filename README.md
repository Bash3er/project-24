# Course Management System

A comprehensive Angular-based application designed to streamline course management and student enrollment processes for educational institutions.

## Description

This application provides an intuitive interface for administrators and educators to manage courses, track student enrollments, and monitor system performance through a centralized dashboard. Built with modern web technologies, it ensures scalability, maintainability, and a seamless user experience.

## Prerequisites

Before running the application, ensure you have the following installed:

- Node.js (version 18 or higher)
- npm (comes with Node.js)
- Angular CLI (version 17 or higher)

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd project-24
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the mock backend server:
   ```bash
   npm run json-server
   ```

2. In a separate terminal, start the Angular development server:
   ```bash
   ng serve -o
   ```

The application will be accessible at `http://localhost:4200`.

## Features

- **Course Management**: Create, view, and manage course details.
- **Student Management**: Maintain student records and information.
- **Enrollment Tracking**: Monitor and manage student enrollments in courses.
- **Dashboard Overview**: Get insights into system metrics and performance.

## Tech Stack

- **Frontend**: Angular 17
- **UI Library**: Angular Material
- **Backend**: JSON Server (for mock data)
